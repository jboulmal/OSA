ATTENTION: Using additional apache configuration directives may corrupt entire configuration if directives are not valid...
Addtional apache directive:
In addition to standard apache variables, refering to the node where request is handled, you may use here:";
  • %{publicServerProtocol}e as protocol used (i.e http:// or https://)\n"
  • %{publicServerName}e server name\n"
  • %{publicServerPort}e server port\n"
  • %{frontEndEndPoint}e frontEndPoint\n"
  • %{publicServerPrefix}e concatenation of previous variables (ex: https//public.node.com:8443/myservice)\n"

    • Ex:
    RequestHeader set Public-Root-URI "%{publicServerProtocol}e%{publicServerName}e:%{publicServerPort}e/%{frontEndEndPoint}e"";
